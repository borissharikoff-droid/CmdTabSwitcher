CmdTabSwitcher — если macOS ругается "нельзя открыть" / "повреждено"
================================================================================

Это не вирус — просто приложение не из App Store и не подписано платным
сертификатом Apple (это стоит $99/год), поэтому Gatekeeper блокирует его
по умолчанию у всех, кто не является автором.

СПОСОБ 1 (самый простой):
Дважды кликни файл "Установить.command" в этой папке. Если macOS спросит
что-то вроде "открыть с помощью Terminal?" — нажми "Открыть". Скрипт сам
поставит приложение, снимет блокировку и запустит его.

СПОСОБ 2 (если способ 1 тоже заблокирован — актуально для самых новых
версий macOS, где Apple убрала старый "ПКМ → Открыть"):
1. Попробуй открыть приложение (двойной клик) — появится ошибка, закрой её.
2. Открой  System Settings → Privacy & Security.
3. Прокрути страницу ВНИЗ до раздела с сообщением про CmdTabSwitcher —
   там будет кнопка "Open Anyway" / "Всё равно открыть".
4. Нажми её, введи пароль от Мака если попросит.
5. Попробуй открыть приложение СНОВА (двойной клик) — теперь появится
   диалог с кнопкой "Open" / "Открыть" — нажми её.

СПОСОБ 3 (самый надёжный, через Terminal — работает всегда):
1. Открой Terminal (Программы → Утилиты → Terminal, или через Spotlight:
   Cmd+Пробел, напиши "Terminal", Enter).
2. Вставь одной строкой и нажми Enter:

   xattr -cr /Applications/CmdTabSwitcher.app && open /Applications/CmdTabSwitcher.app

Если приложение уже не в /Applications (лежит на Рабочем столе/в Загрузках) —
сначала перетащи иконку CmdTabSwitcher.app в папку Applications, потом уже
выполняй команду выше.

ПОСЛЕ ЗАПУСКА:
Приложение попросит два разрешения в System Settings — Accessibility и
Screen Recording. Включи оба тумблера напротив CmdTabSwitcher, иначе
Cmd+Tab не будет работать.

Держи Cmd, жми Tab — переключение между окнами (не приложениями) с превью.
